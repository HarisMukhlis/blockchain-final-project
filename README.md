# Blockchain Simulator Web
A campus assignment for simulating blockchain using simple JavaScript and [elliptic.min.js](https://www.npmjs.com/package/elliptic)

## Features include:
* String Hashing (SHA-256)
* Single Block Generator
* Simplified Blockchain System Simulator
* ECC Signature Simulator and Verifier
* Consensus Simulator using 3 user database
