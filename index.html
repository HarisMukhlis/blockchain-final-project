Selamat datang di Website Belajar Blockchain

Cara Penggunaan

Website ini dibuat untuk memberikan pengetahuan tentang cara kerja sistem Blockchain,
melalui simulasi aplikatif yang dapat dicoba secara langsung.

Untuk mengetahui lebih lanjut, buka salah satu tab pada Navbar di atas:

  • Home: Halaman informasi penggunaan web.
  • Hash: Memasukkan input berupa teks, kemudian otomatis mengeluarkan hasil enkripsi SHA-256 dari input tersebut.
  • Block: Satu block terdiri atas Nomor Block dan Data, yang masing-masing dapat diisi lalu dijalankan proses Mining sederhana untuk mendapat nilai Nonce-nya dalam ketentuan tertentu(Memiliki 4 digit angka nol di depan). Terdapat fitur untuk mengatur seberapa cepat proses Mining dijalankan.
  • Block Chain: Simulasi sekumpulan Block dalam satu sistem, yang masing-masing terhubung dengan nilai Previous Hash. Isi data teks, jalankan Mine lalu tambahkan blok baru menggunakan Add Block.
  • ECC Signature: Lakukan Generate Key Pair terlebih dahulu, lalu isi data pesan dengan teks. Setelah ditandatangani, klik Verify untuk mengecek apakah hasil signature tersebut valid atau tidak.
  • Konsensus: Mensimulasikan 3 user dalam satu Blockchain, masing-masing memiliki saldo dan database sendiri, yang mana bisa dikirimkan ke user lain dengan mengatur Penerimanya. Setelah beberapa transaksi dikumpulkan dalam Mempool, jalankan proses mining dan akan keluar blok baru. Blok ini dapat diubah untuk mensimulasikan peretasan, lalu tekan Verify untuk menampilkan blok mana saja yang telah dimanipulasi, lalu Consensus untuk mengembalikannya kembali berdasarkan nilai dari database user lainnya.
